heroku keys:add [KEY]

add a key for the current user

if no KEY is specified, will try to find ~/.ssh/id_[rd]sa.pub

$ heroku keys:add
Could not find an existing public key.
Would you like to generate one? [Yn] y
Generating new SSH public key.
Uploading SSH public key /.ssh/id_rsa.pub... done

$ heroku keys:add /my/key.pub
Uploading SSH public key /my/key.pub... done

  
heroku keys:remove KEY remove a key from the current user $ heroku keys:remove email@example.com
Removing email@example.com SSH key... done		  

heroku keys:clear

 

 remove all authentication keys from the current user $ heroku keys:clear
Removing all SSH keys... done		  

 

 

sharing

 

 

       
heroku sharing list collaborators on an app $ heroku sharing
=== example Collaborators
collaborator@example.com
email@example.com		  
heroku sharing:add EMAIL add a collaborator to an app $ heroku sharing:add collaborator@example.com
Adding collaborator@example.com to example collaborators... done		  
heroku sharing:remove EMAIL

remove a collaborator from an app

 $ heroku sharing:remove collaborator@example.com
Removing collaborator@example.com to example collaborators... done		  
heroku sharing:transfer EMAIL transfer an app to a new owner $ heroku sharing:transfer collaborator@example.com
Transferring example to collaborator@example.com... done
