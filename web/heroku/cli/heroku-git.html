heroku help git

Additional commands, type "heroku help COMMAND" for more details:

git:clone APP [DIRECTORY] # clones a heroku app to your local machine at DIRECTORY (defaults to app name)
git:remote [OPTIONS] # adds a git remote to an app repo

  
heroku help git:remote

Usage: heroku git:remote [OPTIONS]

adds a git remote to an app repo

if OPTIONS are specified they will be passed to git remote add

-r, --remote REMOTE # the git remote to create, default "heroku"

Examples:

$ heroku git:remote -a example
Git remote heroku added

$ heroku git:remote -a example
! Git remote heroku already exists

  
heroku help git:clone

clones a heroku app to your local machine at DIRECTORY (defaults to app name)

Examples:

Cloning from app 'example'...
