installing

\curl -L https://get.rvm.io | bash -s stable --ruby

or

\curl -#L https://get.rvm.io | bash -s stable --autolibs=3 --ruby

 notes: no sudo  
       
making effective - after installing

on opening terminal, run source /Users/juguang/.rvm/scripts/rvm

on newly opened terminal. do nothing. since ~/.bash_profile has been executed.

Following line is added into ~/.bash_profile during installation

[[ -s "$HOME/.rvm/scripts/rvm" ]] && source "$HOME/.rvm/scripts/rvm" 
# Load RVM into a shell session *as a function*
