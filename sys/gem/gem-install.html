gem-install

 

 

       
$ gem install rails 4.0.5 ERROR: Could not find a valid gem 'rails' (>= 0), here is why:
Unable to download data from https://rubygems.org/ - no such name (https://rubygems.org/latest_specs.4.8.gz)
ERROR: Could not find a valid gem '4.0.5' (>= 0), here is why:
Unable to download data from https://rubygems.org/ - no such name (https://rubygems.org/latest_specs.4.8.gz)		    
$ gem install rails --version=4.0.5 ERROR: Could not find a valid gem 'rails' (= 4.0.5), here is why:
Unable to download data from https://rubygems.org/ - no such name (https://rubygems.org/specs.4.8.gz)		    
       


 

Founding:

  • If you specify a version incorrectly, the version will be treated as a gem.
  • If you did not specific a version, gem will load https://rubygems.org/latest_specs.4.8.gz
  • If you specify a version, gem will load https://rubygems.org/specs.4.8.gz (attention: no latest in url)
  • If the error is like above, it is most probably that your machine does not connect to the Internet properly.
