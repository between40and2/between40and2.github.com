JXAction, JXUIAction and JXUIVCAction

JXUIAction family is descendant of this JXAction

 

  JXAction JXUIVCAction  
       
 
  • it serves a certain object. working around the object
  • it represent an action around the object.
  • actions, which share the same object, can be grouped into list
  • it can be subclassed or configured.
  • it takes in view controller, as a target and resource
  • it provides a buttonItem, type of UIBarButtonItem, ready to be used on navigationItem or toolbar
  • for list, it provides an actionsheet. The actionsheet will be presented according to the user idiom of device (e.g. iPhone or iPad)
